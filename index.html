THE MAGIC NUMBER GAME

RULE

The game will randomly generate any number called the MAGIC NUMBER between 0 and 100, the player must guess that number by puting inside the following input, HE HAS ONLY 7 ATTEMPTS, if he loses he can click the RESTART button for one more chance.
All the best !!!

Number of atempt: 7
